# Cyberpunk Tetris 🎮

Современная версия классического тетриса, реализованная на чистом JavaScript с киберпанк-стилистикой и адаптацией под мобильные устройства.

## 🚀 Возможности

* Классический геймплей Tetris
* Предпросмотр следующей фигуры
* Система очков, уровней и рекорда
* Анимация очистки линий
* Пауза / продолжение игры
* Сохранение рекорда (LocalStorage)
* Полностью адаптивный интерфейс (desktop + mobile)
* Поддержка сенсорного управления

## 📱 Мобильная версия

* Управление через экранные кнопки
* Поддержка удержания кнопок (движение фигуры)
* Tap по игровому полю:

  * старт игры
  * пауза / продолжение
* Верхняя панель с:

  * счётом
  * уровнем
  * линиями
  * следующей фигурой

## 🕹 Управление (ПК)

* ← → — движение
* ↑ — вращение
* ↓ — ускоренное падение
* Space — мгновенное падение
* P — пауза

## 📱 Управление (мобильные)

* Кнопки на экране
* Удержание = непрерывное движение
* Тап по экрану:

  * старт
  * пауза
