# Cyberpunk Tetris 🎮

A modern Tetris game built with pure JavaScript, styled in a cyberpunk aesthetic.

## 🚀 Features
- Classic Tetris gameplay
- Next piece preview
- Line clear animation
- Score, level, and high score system
- Pause / resume
- LocalStorage support
- Responsive layout

## 🕹 Controls
- ← → — move
- ↑ — rotate
- ↓ — soft drop
- Space — hard drop
- P — pause
