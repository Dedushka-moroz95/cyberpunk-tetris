# Cyberpunk Tetris

Адаптивный тетрис на чистом HTML, CSS и JavaScript. Игра сделана без сборки и внешних библиотек: достаточно открыть `index.html` в браузере.

Проект ориентирован на аккуратный desktop-интерфейс и удобную игру в мобильном браузере: есть сенсорные кнопки, compact HUD, очередь фигур, пауза, экран завершения и визуальная обратная связь.

## Возможности

- Классическое поле 10 x 20.
- Генерация фигур через 7-bag, чтобы последовательность была честнее и предсказуемее.
- Очередь из четырех следующих фигур.
- Ghost piece для предварительного просмотра места падения.
- SRS-повороты с wall kicks.
- Lock delay: фигура не фиксируется мгновенно при касании пола.
- Счет, линии, уровни и лучший результат в `localStorage`.
- Стартовый экран, пауза и экран завершения игры поверх поля.
- Анимация очистки линий.
- Визуальная обратная связь для hard drop, очистки линий, тетриса и нового рекорда.
- Вибрация через Vibration API там, где браузер ее поддерживает.
- Визуальный fallback для iOS Safari, где Vibration API недоступен.
- Адаптивная верстка для desktop и мобильных браузеров.

## Управление

### Клавиатура

- `←` / `→` — переместить фигуру.
- `↑` — повернуть фигуру.
- `↓` — мягкое падение.
- `Space` — hard drop.
- `P` — пауза.
- `Enter` — старт, рестарт после завершения или продолжение с паузы.

### Сенсорное управление

На мобильных экранах появляются кнопки:

- `←` / `→` — перемещение.
- `↻` — поворот.
- `↓` — мягкое падение.
- `⇩` — hard drop.
- `⏸` / `▶` — пауза и продолжение.

Кнопки перемещения и мягкого падения поддерживают удержание.

Дополнительно можно управлять жестами по игровому полю:

- тап — поворот;
- свайп влево или вправо — перемещение;
- короткий свайп вниз — мягкое падение;
- длинный свайп вниз — hard drop;
- свайп вверх — поворот.

## Мобильная адаптация

Мобильный интерфейс использует компактный HUD над полем:

- счет;
- уровень;
- линии;
- следующая фигура;
- отдельная кнопка паузы.

Разметка учитывает `viewport-fit=cover` и безопасные зоны экрана. Для iOS Safari вибрация заменяется визуальным тактильным откликом, потому что браузер не поддерживает `navigator.vibrate()`.

## Игровая логика

- Размер поля: 10 колонок и 20 строк.
- Фигуры выбираются из перемешанного набора из 7 фигур.
- Скорость увеличивается с уровнем.
- Уровень растет каждые 10 очищенных линий.
- Очки начисляются за линии, мягкое падение и hard drop.
- Лучший результат сохраняется локально в браузере.

## Запуск

Откройте `index.html` в браузере.

Локальный сервер не обязателен, потому что проект не использует модули, сборщик или сетевые запросы.

## Структура проекта

.

## Технологии

- HTML5 Canvas.
- CSS Grid/Flexbox.
- Vanilla JavaScript.
- `localStorage` для сохранения рекорда.
- Pointer Events для сенсорного управления.
- Vibration API с визуальным fallback.
